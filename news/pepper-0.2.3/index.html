pepper/ news/ pepper 0.2.3 released

I'm happy to announce that version 0.2.3 of pepper is now available. This release adds the punchcard report, inspired by GitHub's punchcard and Git Timecard. Additionally, a major glitch preventing the Subversion backend to work on empty caches has been fixed (#3314428, thanks to Daniel Lynes for reporting!).

A more detailed listing of the changes can be found in the change log. As usual, source tarballs can be found at the Download section.
